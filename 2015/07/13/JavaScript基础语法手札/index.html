1.用户界面：让用户可以看到网页的内容

1.读取html文件，根据html中每个标签之间的相互关系生成一个树模型

2.读取css文件，根据csss中每个样式，生成一个样式树模型

3.将结构树和样式树进行结合，生成渲染树（页面准备显示了）

4.根据渲染树生成网络页面

> 作用域

  JS中只有函数可以分隔独立的作用域

  作用域中声明的变量只能被内部使用


  全局作用域：

      Script内部，不在任何函数内部

  局部作用域：（函数作用域）

      在某函数内部，称为局部作用域


  当我们在某作用域下使用一个变量时

      1.会在当前作用域查找该变量的声明

      2.如果没有找到，会向父级作用域查找

      3.如果没有找到继续上访直至全局变量

      4.如果还没有找到

        使用方式为取值则直接报错

            console.log（ num ）；

        使用方式为设置则在全局作用域下声明一个全局变量

            num = 10；



> 预解析     声明提前化

    2.查找当前作用域中的fn函数声明语句，将函数体提升到当前作用域

对象 
  js中的对象用于描述一个具体的事物

      每个事物都具有一些特征，称为对象的属性

      每个事物还具有一些行为，称为对象的方法


  常规创建:

      var obj = new Object();

  属性赋值：

      obj.name = "亚瑟"；

      obj.age = "26"；

      调用obj.name;


  方法添加;

      obj.add = function(){ }；

      调用obj.add();


      方法中的this:

      object.add = function(){ this }

      this指向函数的调用者Object，函数属于谁




对象创建剖析：

  将以下函数名改为Object即为常规构造函数创建方式

    var obj = new Object(name，age); //传参

    function Object( name ){    //构造函数

        this.name = name;

        this.age = age;

    }


    new的作用：

       Stop-1. new帮我们创建对象

       Stop-2. 将this指向修改为创建的对象

       Stop-3. 返回此对象


  构造函数内不需要设置返回值，如果设置返回值

    基本类型：会被无视

    复杂类型：会替换new新建的对象新建

正则表达式 
  又名规则表达式，由基本的元字符构成

  作用：匹配字符串的


  元字符

       .   表示除\n以外的任意单个字符

       []    表示该范围内的单个值

            [a]  [a-z]  [0-9]  [0-9a-zA-Z]

       |     或者

            [0-9] | [0-9] 


      （）   提升优先级

            （[a-z] | [0-9]）[A-Z]



  限定符

      +   表示这个符号前面的表达式，最少出现1次

            [0-9][a-z] ---- "4serl"

      *   表示这个符号前面的表达式，最少出现0次

            [a-zA-Z][0-9]* ---- "b2018" ---- "n"

      ?   表示这个字符前面的表达式，出现0-1次

            [0-9]? ---- "6" ---- ""

    {1,3}   表示这个字符前面的表达式，出现1-3次

            {1，} ---- 至少出现1次


      \d    表示单个数字，相当于[0-9]
      \D    表示单个非数字

      \s    表示单个空白符号
      \S    表示单个非空白符

      \w    表示单个非特殊符号
      \W    表示单个特殊符号@#%

      ^   以何开头  ^[0-9]
          取非    [^0-9]

      $   以何结尾  [a-z]$





  创建正则表达式对象

    通多构造函数创造对象

    通多字面量方式创建对象

    调用方法测试

    括号分组正则表达式,调用exec方法转化为数组

    var reg = /(\d{2,3}) \: (\d{2,3})/

    var arr = reg.exec("00:42") ----  ["00", "42"]
