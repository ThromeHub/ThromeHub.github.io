Browser Object Model —- 浏览器对象模型 
      作用：将浏览器的功能设置为对象形式，提供给我们一些属性与方法使用

          BOM中的顶级对象：window对象

          也是JS在浏览器环境中的顶级对象


      通常情况下，在使用whindow的属性或方法时可以省略window

          如window.alert();       只有按钮

            window.confirm();     有确认及取消按钮，返回bool

            window.prompt();      可返回输入内容


      所有生命的全局变量都是window的属性

          var num = 10;   

          window.num ---- 10

Location对象 
  location是window的属性  

      href属性 ---- 域名

          location.href = "index.html"

          表示跳转到./域名


      assign方法  

          location.assign("index.html")

          跳转到设置的链接


      replace方法

          location.replace("index.html")

          可以跳转到设置的链接，无法回头

      reload方法

          location.reload(true)

          刷新页面

          如果不设置参数会读取缓存记录

单线程与多线程 
  1.js是单线程的

        H5中引入了多线程的概念

        但是还是只有唯一主线程运作，其他线程辅助

  2.单线程指的就是只有一人做事情


      单线程问题：

          一次仅能做一件事，如果某件任务比较耗时间

          这时会对后边任务造成阻塞
同步与异步 
  同步：一人做事，按照顺序做事情，中间不能做别的事情

  异步：当一个任务比较耗时，或者执行时间不稳定，这时会设置为异步任务


        常见异步任务：事件，定时器，Ajax


  当程序执行时

        发现同步任务

            将这个任务直接放入到主线程中进行排队

    发现异步任务

          1.将这个异步任务不会直接添加到主线程中进行排队，而是添加到任务队列中

          2.当任务队列中的某个任务满足条件时，将需要执行的代码添加到主线程中

          所有的异步任务的执行均晚于同步任务
