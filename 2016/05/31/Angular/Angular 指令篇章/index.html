Angularjs常用指令

指令分类:

  • 内置指令：普通指令 - 事件指令

普通指令

ng-repeat指令: 遍历并输出数据

  • 遍历数组：data

  • 遍历数组对象：person

  • 遍历对象：obj

  • 初始化数据：data person obj

ng-repeat:遍历过程中的内置参数

  • $index : 遍历过程中的索引

  • $first : 布尔类型的值 当前是否是第一项 是返回true 不是返回false

  • $last : 布尔类型的值 当前是否是最后一项 是返回true 不是返回false

  • $middle : 布尔类型的值 第一项和最后一项返回false 其余返回true

  • $even : 偶数项返回true 否则返回false

  • $odd : 奇数项返回true 否则返回false

  • 实例说明：利用奇偶项控制首尾两项样式

ng-repeat:重复参数输出问题

  • 默认情况下遍历的数组中不能有重复类型的值(基本数据类型) 否则会报错

  • 解决: 在遍历的数据后面写追加 track by $index

ng-bind : 数据绑定 解决表达式闪烁问题

  • 原理：在angular对视图更新之前，浏览器不能识别括号从而导致闪烁问题的出现

　　　　　采用bind指令系列的原理在于，浏览器识别自定义属性，而属性不会展现在视图中

  • 语法：ng-bing = “变量名”

  • 原理与ng-bind类似，区别在于支持多变量绑定

ng-cloak: 数据绑定 解决表达式闪烁问题

  • 原理：angular会隐藏含有ng-cloak样式和指令的元素并在解析完成后移除元素的ng-cloak样式和指令从而解决表达式闪烁的问题。不过angular在页面的最后才被引用进来,所以不能很好的解决这个问题需要我们手动在页面顶部加上隐藏元素的样式[ng-cloak]{ display:none }

ng-show ng-hide ： 显示/隐藏元素 样式操作（display）

  • ng-show ：控制元素显示隐藏(通样式的方式)

    • 当值为true时 元素显示

    • 当值为true时 元素隐藏
    • 当值为false时 元素显示

自定义指令

参考：http://www.cnblogs.com/webHero/p/5770703.html

需求：angularjs官方提供的常用指令无法满足实际工作中的需求

用户需要自定义指令实现一些特定需求  ( 轮播图 回到顶部 吸顶导航.. )

作用:

  • 1.提供给我们操作Dom的方式

  • 2.封装一些[可复用]的[UI组件]

调用

指令的配置参数

restrict 自定义指令调用方式设定

  • 默认设置：restrict:”AE”

  • 属性指令A attribute

  • 元素指令E element

  • 样式指令C class

  • 注释指令M comment

template / templateUrl 替代指令标签的文本内容

  • template 模板字符串

    template:"<div>{{ data }}</div>"

  • templateUrl 请求数据模板

    templateUrl:"./template.html"

  • templateUrl 调用angular定义模板

    templateUrl:"模板ID"

replace: 模板根节点的解除包裹操作

  • replace:true

    • 将当前指令所在的标签替换掉

    • 模板内需要唯一根节点继承指令属性，否则会报错

    • 布尔类型的值 默认是是false 意为不替换

transclude: 保留/选用指令标签原有内容

  • transclude：true

    将指令原有内容保存到模板内含有ng-transclude的标签内

    默认值是false 不保存原有内容 也就是直接用模板内容替换原有内容
