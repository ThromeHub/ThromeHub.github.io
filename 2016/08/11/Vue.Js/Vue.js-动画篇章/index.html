配合animate.css

需要条件

  • 1.动画元素有隐藏消失的效果触发

  • 2.动画元素引入类库class=”animated”

  • 3.transition标签引入特定类

    enter-active-class；动画进入时的效果类

    leave-active-class：动画离开时的效果类

标签:

注意：需要绑定：key属性进行区别

配合v-for使用

动画钩子函数

 <transition enter-active-class='zoomInLeft' leave-active-class='zoomOutRight'
                @before-enter = ""
                @enter = ""
                @after-enter = "afterEnter"
                @before-leave = ""
                @leave = ""
                @after-leave = "afterLeave"
>

<script>
    new Vue({
        methods:{
            afterEnter(element){ },            //指定动画进入完成时执行        
            afterLeave(element){ }            //指定动画离开完成时执行
        }
    })//element参数对应动画DOM元素
</script>
