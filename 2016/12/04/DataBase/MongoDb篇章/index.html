MongoDB

数据库概念

存储优势：

  • 相比文件存储更加安全，更加结构化。

  • 相比内存存储数据，不会在重启后丢失

数据库自身也是程序

  • 公司环境：机房里面有服务器，服务器连接数据库

  • 本机操作：可以连接服务器，也可以直接连接数据库

数据库程序结构：客户端 + 服务器

  • 服务器可对应多个客服端

数据库简史

基本操作类型：CRUD —- 增删改查
SQL结构化查询语句：

  • 查询方式：select * from 表名

  • 特点：

    • 各个数据结构之间有强烈关联，在使用前要说明其关联，声明数据类型.有对应的查询语句来获取数据

    • oracle 主流数据库

    • sqlserver 早期只支持windows平台，对.net支持较好，现在也能够跨平台

    • mysql 瑞典一家公司开发的开源的数据库,已被oracle收购

Nosql非关系型：Mongodb redis
商业及个人版本：个人版本限制大小及配置

Mongodb客户端及服务器

安装：npm install mongodb registry

验证：mongo –version 版本信息

链接：mongo IP/数据库名 ( > ok )

MONGODB数据库结构关系

数据库程序 —- 多个数据库 show dbs

数据库 —- 多个集合 show collections

集合 —- 多个文档对象 对应查询语句

基本操作

切换数据库：use 数据库名

集合名：users

插入数据：

  • db.集合名.insert(对象)

  • db.users.insert({ name:”melody” })

  • 提示：WriteResult({ “nInserted” : 1 })

修改数据：

  • db.集合名.update({ 条件对象 },{ $set:{修改对象} })

  • db.users.update({ name:”melody” },{ $set:{ name:”Sakurio”,age:24 }})

  • 提示:WriteResult({ “nMatched” : 1, “nUpserted” : 0, “nModified” : 1 })

查询数据：

  • db.集合名.find({ 条件对象 })

  • db.users.find({ age:18 })

  • 唯一查询方式：db.users.find({_id:ObjectId(“59ae655fa499b98de6452ae0”)})

删除数据：

  • db.集合名.remove({ 条件对象 })

  • db.users.remove({_id:ObjectId(“59ae655fa499b98de6452ae0”)});

  • 提示: WriteResult({ “nRemoved” : 1 })

条件查询

  • “>” - $gt db.users.find({ score:{ $gt:60 }})

  • “<” - $lt db.users.find({ score:{ $lt:90 }})

  • “>=” - $gte db.users.find({ score:{ $gte:60 }})

  • “<=” - $lte db.users.find({ score:{ $lte:90 }})

  • “!=” - $ne db.users.find({ score:{ $ne:80 }})

分页

  • 语法：db.集合名.find( ).skip( 起始索引 ).limit( 获取数量 );

  • 示例：查询学生集合第3到6条

    db.users.find().skip(2).limit(4)

排序

  • db.集合名.find().sort({ 条件:1|-1 }) //正序1 逆序-1`

  • db.users.find().sort({ score:1 }) //正序排列学生成绩

  • 示例：查询成绩大于等于60分，逆序排列取2条数据

    db.users.find({score:{$gte:60}}).limit(2).sort({ score:-1 })

模糊匹配

  • db.users.find({ name:正则 })

  • db.users.find({ name:/^路/ }) //匹配路开头的数据

  • db.users.find({ name:/路$/}) //匹配路结尾的数据

  • db.users.find({ name:/路/}) //匹配涵盖路的数据

获取查询文档对象数量

  • db.集合名.find({条件}).count() —- num

  • db.users.find({score:{ $ne:88 }}).count() —- 45

  • 示例：查询成绩不等于88分，并且名称中包含路字的总数

    db.users.find({ score:{ $ne:88 },name:/路/ }).count() —- 2

  • 并且条件用逗号分离：

    db.users.find({ score:{ $ne:88 },name:/路/   })
