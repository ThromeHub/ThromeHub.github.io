组件加载时

| getDefaultProps(){}        

    组件创建或父组件属性传入前被调用，准许我们指定this.props默认值


| getInitialState(){}        

    组件创建前调用，准许我们指定this.state默认值

    只在React.createClass方法创造的组件内可用，对于类声明的组件不适用


| componentWillMount(){}    

    组件被渲染到DOM前调用，此阶段无法通过setState方法修改状态

    只在React.createClass方法创造的组件内可用，对于类声明的组件不适用


| rander

    将组件内容渲染 返回false或null时不渲染内容


| componentDidMount(){}    

    组件渲染到DOM后调用
